# clock

A simple time tracking tool for use on the command line.

## Installation

```bash
go install github.com/zietzm/clock@latest
```

## Usage

The command line usage is as simple as can be.

```bash
clock in
clock out
```

You can specify a category like

```bash
clock in work
clock out work
```


To view these records, run

```bash
clock log
```
